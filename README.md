# Nuxt 3 + Vuetify 3 Starter

![Starter Image](/public/logo.png)
Nuxt 3 is now stable. Vuetify 3 is now stable. Feel free to open an issue if you have any questions or suggestions.

### Run the app

1. Install the deps

```bash
npm install
# or
yarn install
```

2. Run the app

```bash
npm run dev
# or
yarn dev
```

3. Build the app

```bash
# for ssr single page application
npm run generate
yarn generate

# for ssr mode
npm run build
yarn build
```
